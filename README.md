# NRTLC-A-QUIZ
![Status](https://img.shields.io/badge/Status-In_Sviluppo-orange)
![Field](https://img.shields.io/badge/Ambito-Ingegneria_Telecomunicazioni-blue)

Repository dedicata allo sviluppo di un sistema di valutazione e simulazione d'esame per il corso di **Reti di Telecomunicazione (NRTLC)**. Il progetto mira a fornire uno strumento di autovalutazione sui protocolli, le architetture e il calcolo delle prestazioni di rete.

## 📌 Obiettivi del Progetto
L'applicativo è progettato per testare la comprensione dei seguenti pilastri ingegneristici:
- **Modelli di Riferimento:** Architetture ISO/OSI e stack TCP/IP.
- **Analisi Quantitativa:** Calcolo di Throughput, Goodput e Ritardi (trasmissione, propagazione, accodamento).
- **Protocolli:** Funzionamento di strato applicativo (HTTP, DNS), trasporto (TCP, UDP) e rete (IPv4/v6, Routing).
- **Livello Data Link:** Controllo dell'errore e accesso al mezzo (MAC).

## 🛠 Caratteristiche Tecniche
- **Interfaccia:** Gestione sequenziale dei quesiti con valutazione finale.
- **Data Scoring:** Algoritmo di calcolo del punteggio basato su criteri accademici standard.
- **Modularità:** Struttura predisposta per l'espansione del database dei quiz tramite file di configurazione esterni.
